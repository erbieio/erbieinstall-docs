Getting Started
###############################

If you want to be a validator or staker, please make sure you have at least 350 ERB, otherwise you cannot become a Validator or staker, others please go to `Discord <https://discord.com/invite/N4ksH6tqRX>`__ consultation, Please follow the documents below to complete the deployment and go back to the wallet to finalize the stake.

If you don't have accounts, you can generate your own account through `Erbie Wallet <https://wallet.erbie.io>`__.
